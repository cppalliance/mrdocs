[#sqrt]
== sqrt


Computes the square root of an integral value.


=== Synopsis

Declared in `<pass:[sqrt.cpp]>`
[source,cpp,subs="verbatim,macros,-callouts"]
----
template<typename T>
std::T
sqrt(T value);
----

=== Description


This function calculates the square root of a
given integral value using bit manipulation.
